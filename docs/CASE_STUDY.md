# Case Study: E-VIL Villain Assistant

## Resumen

AI assistant con personalidad villain, integración con Google Search, navegador virtual cyber-terminal y autenticación.

## Problema

Los asistentes AI son genéricos y aburridos. No hay experiencias con personalidad fuerte que enganchen al usuario.

## Solución

Asistente con personalidad villain que combina chat AI (Groq), búsqueda web (Serperdev), navegador virtual embebido, y reconocimiento de voz.

## Stack

Next.js 16 + React 19 + TypeScript + Tailwind CSS 4 + Neon/PostgreSQL + Drizzle ORM + Groq + Serper.dev

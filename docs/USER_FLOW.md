# User Flow

## Visitante

```
Landing → Terminal Console
  ├── Chat with AI (villain personality)
  ├── Web Search (side panel)
  ├── Virtual Browser (embedded viewer)
  └── Login / Register
```

## Autenticado

```
Dashboard
  ├── Chat histórico
  ├── Búsquedas con grounding
  ├── Navegador virtual
  └── Voice input / synthesis
```

## Auth

```
Register → Login → Terminal
  ├── Forgot password → Email token → Reset → Login
  └── Logout
```

# Features

## Implementadas

| Feature | Descripción |
|---|---|
| AI Chat | Terminal interactiva con personalidad villain (Groq) |
| Web Search | Google Search via Serper.dev con panel lateral |
| Virtual Browser | Visor de páginas web embebido con navegación |
| Voice & Synthesis | Speech recognition, selección de voz y velocidad |
| Auth completo | Register, login, forgot/reset password |
| i18n ES/EN | Incluye auth y virtual browser |
| Tutorial onboarding | Tarjetas interactivas en primera visita |
| CSP nonce | strict-dynamic |
| Rate limiting | Chat 10/min, Browser 5/min, Login 10/min |
| Google Analytics | GA4 opt-in via env var, CSP whitelist |
| Sentry | Error tracking con `withSentryConfig`, 3 configs client/edge/server |
